Coda Plugin

Tower allows you to interact with it from Panic's application "Coda".
To enable this you have to install the Coda plugin that is supplied with Tower.

To interact with Tower from Coda

  • Select "Integration..." from the main "Tower" menu. In the following window, choose "Coda Plugin" from the "Integration Topics" table.

  • Click the "Install" button.

  • You will then have a new menu item named "Tower" in the "Plug-ins" main menu in Coda.
