Command Line Tool

Tower allows you to interact with it right from your console prompt. To enable this you need to install the "gittower" command line tool that is supplied with Tower.

To install the "gittower" command line tool

  1. Select "Integration..." from the main "Tower" menu. In the following window, choose "Command Line Tool" from the "Integration Topics" table.

  2. Click the "Install" button to enable terminal usage.

To interact with Tower from the command line

The following operations are supported from the command line:

  • Opening a repository, in either the "History" or the "Browse" view

  • Adding one ore more existing repository to Tower's Dashboard

  • Starting to initialize / create a new repository

NOTE:For a detailed description of the available commands type "gittower --help" on your command line.
