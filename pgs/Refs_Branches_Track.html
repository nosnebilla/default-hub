Tracking a branch

A local branch can "track" another (usually remote) branch. This sets up a default so the local branch knows which branch to push to and pull from.

To track a branch

Perform one of the following actions:

  • In the Sidebar drag and drop the (presumably remote) branch you want to track onto the local branch, while keeping the CTRL key pressed.

  • In the Sidebar right-click the branch you want to track and select "Track This Branch..." from the contextual menu.

NOTE:When creating a new branch in Tower you will find an option to directly set up a tracking in the corresponding dialog.

NOTE:When publishing a local branch you will find an option to directly set up a tracking in the corresponding dialog.
