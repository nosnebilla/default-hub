Creating a new local tag

Tower allows creating a new local tag in a couple of ways.

To create a new tag

Perform one of the following actions:

  • In the Sidebar drag and drop a local branch on the "Tags" section header. The new tag will be pointing at the dragged branch's current commit.

  • In the Sidebar right-click on the "Tags" section header or click on the "plus" button at the bottom and select "Add New Local Tag...".

  • In the History "List" view right-click on a commit item and select "Create Tag for This Commit...".

  • In the History "Recent" view or in a commit's details view click on the option gear button and select "Create Tag for This Commit...".

TIP:If you enter a description for the new tag this will be shown as the content of the tooltip when hovering over the item in the Sidebar.
